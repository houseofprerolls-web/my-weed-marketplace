## Troubleshooting

### "Invalid login credentials"
**Solution:** The demo account needs to be created in Supabase first. See `DEMO_ACCOUNTS.md` for setup instructions.

### "Cannot access vendor dashboard"
**Solution:** Make sure you're logged in as a vendor account (GreenLeaf, not Customer).

### Page not loading
**Solution:** Check that you have an active internet connection and Supabase is configured properly.

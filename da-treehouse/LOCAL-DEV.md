# Running the Next 16 app (`da-treehouse`)

You are inside the **optional** Next 16 app. For **GreenZone Bolt**, use **`../greenzone-bolt/`** (or `npm run dev` from the repo root).

```bash
cd da-treehouse
npm install
npm run dev
```

Open **http://localhost:3000** (use `-p 3001` if GreenZone is already on 3000).
